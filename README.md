# ActiveBrain

## ¿Cómo Editar X cosa?

1.  Escoger branch a ediar ej. `unity`
2.  Clonar esa branch

Así es como se clona una branch especifica:

``` bash

git clone --branch <nombreBranch> https://github.com/josecarlosmemo/activeBrain.git
```

3.  Hacer los cambios
4.  Profit!
